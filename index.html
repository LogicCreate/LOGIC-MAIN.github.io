LogicPad

A Windows-first visual programming platform designed for clarity, reliability, and scale.

Key Features

Visual Node-Based Editor

Design workflows using an intuitive node-based interface. Connect logic visually to improve readability and maintainability.

Rich Built-in Node Library

Includes nodes for file I/O, text processing, conditions, loops, timers, and variable handling out of the box.

JavaScript Execution Nodes

Embed custom JavaScript logic directly into workflows for advanced processing and flexible automation.

Excel Automation

Create worksheets, read and write cells, and execute macros to automate spreadsheet-based operations.

REST API Integration

Send HTTP requests and integrate with external services using REST API nodes.

Email & Messaging Integration

Send emails and post messages to platforms such as Microsoft Teams and Slack from workflows.

HTML & Web Interaction

Built-in HTML browser nodes enable web content loading and basic interaction within workflows.

Workflow Serialization

Workflows are saved in a structured format to support versioning, compatibility checks, and future extensibility.

Commercial-Grade Stability

Designed for long-term use with a focus on stability, predictable behavior, and continuous improvement.

Release History

Early Beta (2021 – 2023)

  • Initial prototype of a node-based visual workflow editor.
  • Core canvas, grid rendering, node placement, and connection mechanics established.
  • Introduction of variables, property dialogs, and basic execution flow.
  • Early file, math, condition, and timing nodes implemented.
  • Undo/Redo, drag-and-drop, zooming, and basic UI interactions stabilized.
  • Multiple product renames and architectural refactoring to prepare for extensibility.

Public Preview & Feature Expansion (2024)

  • Major UI/UX refinements including DPI awareness and multi-monitor support.
  • Expanded node library: file operations, image processing, browser automation, and scripting.
  • Improved variable management, comments, grouping, and editor productivity features.
  • Stability improvements, performance tuning, and installer refinements.

Advanced Automation & Integration (2025)

  • Excel automation nodes for reading, writing, macro execution, and image embedding.
  • REST API, HTTP communication, and external service integration.
  • AI-related nodes including OpenAI-compatible, Claude, Gemini, Grok, and DeepSeek integrations.
  • Email, messaging, and social platform posting nodes.
  • Workflow serialization, debugging enhancements, and reliability improvements.
  • Ongoing UI polish, configuration dialogs, and workflow creation optimizations.
